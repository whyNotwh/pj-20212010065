#### 微服务开发实践报告

#### 一、实验相关信息
1.语言Python，Mysql
2.实验环境：Pycharm，Navicat
3。技术点：flask框架

#### 二、使用说明
1.URL localhost:5000/api/v1/student
2.可使用postman进行post，get，delete，put等HTTP方法的选择来实现学生信息的增删改查。

### 三、实验过程
1.建数据库
2.使用pymysql连接本地数据库
3.实现增删改查数据
4.实现增删改查各个接口功能
5.安装postman测试接口功能
5.安装docker
6.编写dockerfile并测试运行
